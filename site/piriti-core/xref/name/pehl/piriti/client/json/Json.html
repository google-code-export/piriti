21  /**
22   * Annotation for mapping JSON data to POJO properties. The JSON data is
23   * selected by a "path" expression. In the simplest case the path is just the
24   * key of the JSON data. For more complex cases, you can also use a <a
25   * href="http://code.google.com/p/jsonpath/">JSONPath expression</a>. If
26   * necessary, the selected data is converted to the type of the annotated
27   * property. For some types you can specify a format and a custom converter
28   * which is used to parse / serialize the JSON data to / from the properties
29   * type.
30   * <p>
31   * The annotation must be placed on a field in the POJO or must be part of the
32   * {@code @}{@link JsonMappings} annotation. When placed on a field and the
33   * field is not accessible, Piriti tries to use setters / getters instead.
34   * <p>
35   * The following types are supported:

37   * <tr>
38   * <th>Type</th>
39   * <th>Default path expression</th>
40   * <th>Format options</th>
41   * <th>Converter options</th>
42   * </tr>
43   * <tr>
44   * <td>boolean, Boolean</td>
45   * <td>&lt;propertyname&gt;</td>
46   * <td>No format supported. If specified it is ignored.</td>
47   * <td>No custom converter supported. If specified it is ignored.</td>
48   * </tr>
49   * <tr>
50   * <td>byte, Byte</td>
51   * <td>&lt;propertyname&gt;</td>
52   * <td>No format supported. If specified it is ignored.</td>
53   * <td>No custom converter supported. If specified it is ignored.</td>
54   * </tr>
55   * <tr>
56   * <td>char, Character</td>
57   * <td>&lt;propertyname&gt;</td>
58   * <td>No format supported. If specified it is ignored.</td>
59   * <td>No custom converter supported. If specified it is ignored.</td>
60   * </tr>
61   * <tr>
62   * <td>java.util.Date</td>
63   * <td>&lt;propertyname&gt;</td>
64   * <td>If no format is specified a
65   * {@linkplain name.pehl.piriti.client.converter.DateConverter#DEFAULT_FORMAT

84   * {@link JSONString} and the format must be a valid number format as described
85   * by {@link NumberFormat}</td>
86   * <td>Custom converter supported (JSON data is expected to be a JSON string)</td>
87   * </tr>
88   * <tr>
89   * <td>int, Integer</td>
90   * <td>&lt;propertyname&gt;</td>
91   * <td>If no format is specified the JSON data is expected to be a
92   * {@link JSONNumber}, otherwise the JSON data is expected to be a
93   * {@link JSONString} and the format must be a valid number format as described
94   * by {@link NumberFormat}</td>
95   * <td>Custom converter supported (JSON data is expected to be a JSON string)</td>
96   * </tr>

98   * <td>long, Long</td>
99   * <td>&lt;propertyname&gt;</td>

101  * {@link JSONNumber}, otherwise the JSON data is expected to be a

103  * by {@link NumberFormat}</td>
104  * <td>Custom converter supported (JSON data is expected to be a JSON string)</td>
105  * </tr>
106  * <tr>
107  * <td>short, Short</td>
108  * <td>&lt;propertyname&gt;</td>

111  * </tr>
112  * <tr>

126  * <td>&lt;propertyname&gt;</td>

10  /**
11   * Interface for converting JSON data to an instance of T or a list of Ts. The
12   * implementation for this interface is generated using deferred binding. All
13   * fields of T which are annotated with {@link JsonField} are handled by the
14   * generated JsonReader implementation. If the path expression behind
15   * {@link JsonField} returns some data != null the relevant field is assigned
16   * with the converted value, otherwise the field remains unchanged.
17   * <p>
18   * Please note that the annotated fields in T must not be private!
19   * <p>
20   * The setup of the XmlReader is inspired by the UiBinder and is typically
21   * specified as an inner class:
22   * 
23   * <pre>
24   * pubilc ModalGirlfriendMediator
25   * {
26   *     interface Reader extends JsonReader&lt;ModalGirlfriendMediator&gt; {}
27   *     public static final Reader JSON = GWT.create(Reader.class);
28   *     
29   *     // The fields of this POJO annotated with JsonField.
30   * }
31   * </pre>
32   * 
33   * JSON data can then be converted to an instance of ModalGirlfriendMediator by
34   * calling
35   * 
36   * <pre>
37   * String jsonString = ...;
38   * ModalGirlfriendMediator mgm = ModalGirlfriendMediator.JSON.read(jsonString);
39   * </pre>
40   * 
41   * @param <T>
42   *            The type
43   * @author $LastChangedBy: harald.pehl $
44   * @version $LastChangedRevision: 46 $
45   */
46  public interface JsonReader<T>
47  {
48      /**
49       * Convert the specified JSON string to an instance of T according to the
50       * annotated fields in T. The JSON string should represent a valid JSON
51       * object with key/value pairs. Please note that the JSON string is parsed
52       * using {@link JSONParser#parse(String)} which in turn uses the JavaScript
53       * eval() function. So please *do not* pass an untrusted string into this
54       * method.
55       * 
56       * @param jsonString
57       *            The JSON string used as input. May be <code>null</code>.
58       * @return An instance of T with the mapped JSON data or {@code null} if the
59       *         JSON string was {@code null}.
60       */
61      T read(@Nullable String jsonString);

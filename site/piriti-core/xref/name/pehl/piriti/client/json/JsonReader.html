9   /**
10   * Interface for converting JSON data to an instance of T or a list of Ts. The
11   * implementation for this interface is generated using deferred binding. All
12   * fields of T which are annotated with {@link JsonField} are handled by the
13   * generated JsonReader implementation. If the path expression behind
14   * {@link JsonField} returns some data != null the relevant field is assigned
15   * with the converted value, otherwise the field remains unchanged.
16   * <p>
17   * Please note that the annotated fields in T must not be private!
18   * <p>
19   * The setup of the XmlReader is inspired by the UiBinder and is typically
20   * specified as an inner class:
21   * 
22   * <pre>
23   * pubilc ModalGirlfriendMediator
24   * {
25   *     interface Reader extends JsonReader&lt;ModalGirlfriendMediator&gt; {}
26   *     public static final Reader JSON = GWT.create(Reader.class);
27   *     
28   *     // The fields of this POJO annotated with JsonField.
29   * }
30   * </pre>
31   * 
32   * JSON data can then be converted to an instance of ModalGirlfriendMediator by
33   * calling
34   * 
35   * <pre>
36   * String jsonString = ...;
37   * ModalGirlfriendMediator mgm = ModalGirlfriendMediator.JSON.read(jsonString);
38   * </pre>
39   * 
40   * @param <T>
41   *            The type
